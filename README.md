
# צ'ק מחיר - מדריך למשתמש (Windows)

אפליקציה חכמה להשוואת מחירים בזמן אמת בישראל באמצעות בינה מלאכותית.

## הוראות הפעלה למשתמשי Windows

כדי להריץ את האפליקציה בצורה הקלה ביותר, עקבו אחר הצעדים הבאים:

### 1. הכנות ראשוניות
ודאו שמותקן אצלכם **Node.js**. אם לא, הורידו והתקינו מהאתר הרשמי: [nodejs.org](https://nodejs.org/).

### 2. הפעלת האפליקציה
1. פתחו את תיקיית הפרויקט במחשב.
2. לחצו קליק ימני בתוך התיקייה ובחרו ב-"Open in Terminal" (או פתחו CMD ונווטו לתיקייה).
3. הריצו את הפקודה הבאה כדי להפעיל שרת מקומי מהיר:
   ```cmd
   npx serve .
   ```
4. הדפדפן ייפתח בכתובת: `http://localhost:3000` (או כתובת דומה שתופיע בטרמינל).

### 3. הגדרת מפתח API (חשוב!)
האפליקציה זקוקה למפתח API של Google Gemini כדי לעבוד.
*   ודאו שיש לכם משתנה סביבה בשם `API_KEY`.
*   לחילופין, אם אתם מריצים בסביבת פיתוח, ניתן להוסיף אותו לקובץ הסביבה.

## איך להשתמש באפליקציה?
1. **חיפוש:** הקלידו שם של מוצר (למשל: "קוקה קולה 1.5 ליטר").
2. **המתנה:** המערכת תסרוק את אתרי רשתות השיווק בישראל.
3. **תוצאות:** תקבלו את 5 התוצאות הזולות ביותר שנמצאו, כולל שם החנות והמותג.
4. **חזרה:** בכל שלב ניתן ללחוץ על הלוגו או על כפתור "ראשי" כדי לחזור למסך החיפוש.

## פונקציות מרכזיות
*   **השוואה לפי כמות:** המערכת מנסה לנרמל את המחיר (למשל מחיר ל-100 גרם) כדי לתת השוואה הוגנת.
*   **היסטוריה:** החיפושים האחרונים שלכם נשמרים מקומית לנוחיותכם.
*   **מיקום:** אם תאשרו גישה למיקום, התוצאות יהיו מותאמות יותר לאזור המגורים שלכם.
