
# צ'ק מחיר - מדריך למשתמש (Windows)

אפליקציה חכמה להשוואת מחירים בזמן אמת בישראל באמצעות בינה מלאכותית.

## הוראות התקנה והפעלה למשתמשי Windows

כדי להפעיל את האפליקציה בצורה תקינה, יש להשתמש ב-Vite (כלי פיתוח מודרני). הנה הצעדים הפשוטים:

### 1. הכנות
ודאו שמותקן אצלכם **Node.js**.
אם לא, הורידו והתקינו את הגרסה המומלצת (LTS) מהאתר: [nodejs.org](https://nodejs.org/).

### 2. יצירת מפתח API
1. צרו קובץ חדש בתיקייה הראשית של הפרויקט בשם `.env` (נקודה env).
2. פתחו את הקובץ עם פנקס רשימות (Notepad) והדביקו את השורה הבאה (החליפו את ה-X במפתח שלכם):
   ```
   API_KEY=YOUR_GEMINI_API_KEY_HERE
   ```
   *שימו לב: אין לשים רווחים לפני או אחרי הסימן שווה (=).*

### 3. הפעלה ראשונית
פתחו את שורת הפקודה (Terminal או CMD) בתיקיית הפרויקט והריצו את הפקודות הבאות לפי הסדר:

1.  **התקנת הספריות (פעם אחת בלבד):**
    ```cmd
    npm install
    ```
    *זה ייקח דקה או שתיים.*

2.  **הפעלת האפליקציה:**
    ```cmd
    npm run dev
    ```

3.  כעת יופיע קישור בטרמינל (בדרך כלל `http://localhost:5173`).
    לחצו עליו או העתיקו אותו לדפדפן כדי להתחיל להשתמש באפליקציה!

## איך להשתמש באפליקציה?
1. **חיפוש:** הקלידו שם של מוצר (למשל: "קוקה קולה 1.5 ליטר").
2. **תוצאות:** המערכת תסרוק את הרשת ותציג את 5 האופציות הזולות ביותר.
3. **איפוס:** לחצו על הלוגו או על כפתור "ראשי" כדי לחזור להתחלה.

## פתרון תקלות
*   **מסך לבן?** ודאו שהרצתם `npm run dev` ולא פתחתם את הקובץ ישירות בדפדפן.
*   **שגיאות API?** ודאו שקובץ ה-`.env` קיים ומכיל מפתח תקין.
